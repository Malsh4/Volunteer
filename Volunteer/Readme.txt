How to Run?

--Requirements

	Download and Install any local web server such as XAMPP.



--System Installation/Setup

	1. Open your XAMPP Control Panel and start Apache and MySQL.
	2. Extract the downloaded source code zip file.(Web.zip)
	3. Copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.(ex:-C:\xampp\htdocs)
	4. Browse the PHPMyAdmin in a browser. (http://localhost/phpmyadmin)
	5. Create a new database named "volunteer".
	6. Import the provided SQL file. The file is known as "volunteer.sql" located inside the "DataBase" folder.
	7. Browse the System in a browser.(http://localhost/web)
